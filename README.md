# Flight Tracker & Information Display System

## Project Overview
Flight Tracker & Information Display System is a **Java full-stack web application** that displays **arrival and departure flight information** for a selected airport.  
It uses **Spring Boot** for the backend, **MySQL** for storing flight data, and **HTML/CSS/JavaScript** for the frontend.  

This project is designed for beginners to learn **full-stack development** step by step.

---

## Features

### Core Features
- Display **Arrival Flights** with flight number, airline, origin, arrival time, and status
- Display **Departure Flights** with flight number, airline, destination, departure time, and status
- Airport-based flight listing
- REST API backend providing flight data
- Manual database entry for initial setup

### Optional / Future Features
- Search and filter by flight number or airline
- Automatic data updates using flight APIs
- Refresh button for live updates
- Color-coded flight status (On Time / Delayed / Cancelled)
